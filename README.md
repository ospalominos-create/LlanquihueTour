# LlanquihueTour

## Descripción
Proyecto Java orientado a objetos que modela personas vinculadas a la agencia de turismo Llanquihue Tour. Aplica encapsulamiento, composición, herencia y organización por paquetes.

## Estructura del proyecto
- `app`
  - `Main.java`
- `model`
  - `Direccion.java`
  - `Persona.java`
  - `Empleado.java`

## Funcionalidad
El sistema crea objetos de tipo Persona y Empleado, cada uno con una dirección asociada, y muestra su información por consola usando `toString()`.

## Cómo compilar y ejecutar
1. Abrir el proyecto en NetBeans.
2. Verificar que la clase principal sea `app.Main`.
3. Ejecutar el proyecto con **Run Project**.
4. Revisar la salida en la consola.
